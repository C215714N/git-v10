Apuntadores de GIT

HEAD

Apuntador Movil que indica al usuario en que punto se encuentra dentro del historial de confirmaciones, se utiliza como referencia para navegar dentro del mismo y para ultilizar los diferentes de manera relativa.

BRANCH

Apuntador Dinamico que siempre se corresponde con el ultimo commit de la historia que se esta representando, facilitan la navegacion del historial ya que permiten volver al ultimo cambio realizado. Tambien se utilizan para implementar caracteristicas sin comprometer la linea principal.

  • git branch lista las ramas creadas en el repositorio
  • git branch <branch> crea una rama con el nombre indicado
  • git branch -d <branch> elimina la rama si se integraron los cambios
  • git branch -D <branch> elimina la rama aunque no haya sido fusionada
  • git branch -mv <name> renombra la rama actual
  • git push <remote> <branch> envia al repositorio remoto la rama seleccionada

TAG

Apuntador Estatico que se utiliza para el versionado del proyecto, normalmente con nombres como v0.0.0, donde el primer digito correponde a la version principal, el segundo al numero de compilacion, subversion o caracteristica incorporada; y el ultimo para identificar la correccion de errores.

  • git tag muestra las etiquetas asignadas
  • git tag -d <tag> elimina la etiqueta seleccionada
  • git tag <tag> asigna una etiqueta al commit actual
  • git push <remote> --tags envia al repositorio las etiquetas creadas

STASH

Apuntador de la zona temporal de cambios (pila stash) que se utiliza cuando necesitamos cambiar de rama y poseemos cambios pendientes en la actual, pero los mismos aun no han sido finalizados y no amerita la realizacion de un commit. Esta apuntador cuando lo aplicamos realiza una operacion merge a la inversa.

  • git stash agrega los cambios del INDEX a la zona STASH
  • git stash list muestra todas las referencia almacenadas en la zona STASH
  • git stash apply aplica los cambios del STASH en el Working Directory
  • git stash drop elimina la referencia de la zona STASH
  • git stash pop aplica los cambios y elimina la referencia de la zona STASH
